# LK-Mines-2.0-Compatability-Patch
Patches that will add various mod and vanilla resources to Mines 2.0 Bills IF these mods are installed.
Mods Supported
--LK Resources Mod
--Beyond Our Reach
--Outer Rim Core
--Star wars : The Force : Lightsabers
--Grimworld 40k: Core Imperialis
